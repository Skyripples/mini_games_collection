# 小遊戲合集

使用 `HTML`、`CSS` 與原生 `JavaScript` 製作的小遊戲網站，部署於 GitHub Pages。

## 網站

https://skyripples.github.io/mini_games_collection/

## 目前可遊玩的遊戲

- Lv.1 入門：猜數字、迷宮、井字棋
- Lv.2 基礎互動：貪食蛇、2048、打磚塊、打地鼠、記憶遊戲、Flappy Bird、節奏遊戲、倒水分類遊戲
- Lv.3 中階挑戰：踩地雷、打飛機、四子棋、五子棋、推箱子、華容道、數獨、滑塊拼圖
- Lv.4 複合系統：跳棋、俄羅斯方塊、Roguelike、黑白棋
- Lv.5 大型規則：西洋棋、象棋、圍棋、將棋

## 專案結構

- `index.html`：主頁面與各遊戲面板
- `style.css`：整體版型與各遊戲樣式
- `src/main.js`：統一入口與初始化流程
- `src/games/index.js`：遊戲註冊表，集中管理遊戲 `id`、按鈕、面板與建立函式
- `src/games/`：各遊戲模組
- `src/core/`：共用的入口管理、語言切換與主題切換
- `scripts/validate_site.py`：本機靜態驗證腳本

## 本機啟動

```bash
python -m http.server 8000
```

開啟 `http://localhost:8000/` 即可。

## 目前功能

- 主選單可依 `Lv.1` 到 `Lv.5` 篩選遊戲分類。
- 排行榜按鈕會開啟彈出視窗，並可依遊戲與難度分頁查看排名。
- 排行榜支援 GitHub Pages、Live Server 與本機後端來源切換。
- 各遊戲在 `Game Over` 後可送出分數到排行榜。
